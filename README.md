# MyDentalClinic — готовое PWA-приложение

Это простая одностраничная PWA для учёта пациентов в стоматологии (локальное хранилище, экспорт/импорт CSV, офлайн-режим).

Файлы:
- `index.html` — основная страница
- `styles.css` — стили
- `app.js` — логика приложения (localStorage, CRUD, CSV)
- `manifest.json` — манифест PWA
- `sw.js` — service worker для кэширования
- `icon.svg` — иконка (SVG)

Как запустить локально
1. Клонируйте репозиторий или поместите файлы в папку.
2. Для корректной работы service worker нужно запускать через HTTP(S). Например:
   - Быстрый сервер на Python 3:
     - В терминале: `python -m http.server 8000`
     - Откройте http://localhost:8000
   - Или используйте любую другую статическую хостинг-опцию.

Установка как PWA
- Откройте сайт в браузере (Chrome/Edge/Firefox on Android/Safari on iOS — разные уровни поддержки).
- Если браузер поддерживает, появится подсказка об установке (или используйте меню браузера -> «Установить» / «Добавить на главный экран»).

Экспорт / импорт
- Экспорт создаёт CSV-файл с полями: fio,phone,procedure,date,time,duration,price,paid,created.
- Импорт принимает CSV в том же формате.

Развёртывание
- Самый простой способ — включить GitHub Pages для репозитория (Settings → Pages → branch: main / gh-pages).
- Также подойдёт любой статический хостинг (Netlify, Vercel, Firebase Hosting и т.д.).

Дополнительно (рекомендации)
- Замените `icon.svg` на вашу иконку PNG 192×192 и 512×512 для лучшей совместимости.
- Добавьте бэкенд/синхронизацию, если хотите хранить данные на сервере или синхронизировать между устройствами.
- Улучшите обработку CSV (более надёжный парсер) при необходимости.
